This program displays files in your web browser.

To run the program:
    -go into the java files location and open the command prompt 
    -type "javac *.java" to compile
    -type "java WebServer 2225" to run the webserver 
    -to test the program open up your browser and type in "http://localhost:2225/minions.jpg"
    -it will take local files and send them to the webserver
    - you can do it with other files too such as the pdf and text file included
