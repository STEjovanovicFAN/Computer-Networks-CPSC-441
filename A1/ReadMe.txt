This program is an simple socket program requesting data from websites.

List of what it grabs:
    -HTML file 
    -jpeg image
    -pdf
    -text file

you can find all of these in the folder called file folder after you run the program.

To run the program:
    -open a terminal where the java files are located
    -type "javac *.java" to compile the files 
    -to run the program do "java Tester"
